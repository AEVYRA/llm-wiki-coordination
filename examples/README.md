# Examples

This folder will contain illustrative examples once `v0.2`. Planned content:

## `before-after/`

Side-by-side scenarios showing the same handoff with and without the protocol:

- `handoff-without-protocol.md` — three agents working on the same wiki, drift after one month.
- `handoff-with-protocol.md` — the same scenario with the coordination layer in place.

## `sample-thread/`

A complete `dialogue/` thread from `pending-<agent>` through `crystallizing` to `crystallized`, with the resulting workflow page included.

## `sample-consensus-block/`

A real `consensus:` block evolving over four `rev:` revisions, showing stale invalidation in action.

## Contributions

If you adopt this layer in your own wiki and want to contribute a desensitized example, open a PR with the example under this folder and add a corresponding entry above.
